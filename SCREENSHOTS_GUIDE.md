## ğŸ¨ Screenshot Tips

1. **Clean Browser**

   - Hide browser extensions
   - Use incognito/private mode
   - Clean bookmarks bar

2. **Consistent Styling**

   - Same products in screenshots
   - Consistent cart items
   - Same time of day for consistency

3. **Annotations** (Optional)

   - Highlight key features dengan arrows
   - Add captions untuk feature callouts
   - Use consistent color scheme

4. **Tools Recommended**
   - Chrome DevTools untuk responsive testing
   - Lightshot atau Snagit untuk screenshots
   - Figma untuk annotations
   - TinyPNG untuk compression
